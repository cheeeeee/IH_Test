Required Software
1. Node (https://nodejs.org/en/) version 10.1.0 used
2. Selenium Webdriver
3. Nightwatch

Basic Process
1. Download/Clone the files
2. Open CMD/Terminal
3. Change Directory into IH_test
4. npm init
5. npm install selenium and chromedriver
6. npm test

for additional troubleshooting view the markdown file or reference https://github.com/dwyl/learn-nightwatch
